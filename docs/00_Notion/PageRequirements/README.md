### 게스트

- 랜딩 페이지
    - 아이디/비번 입력칸 [로그인버튼]
    - 회원가입 버튼
    - 아이디 기억하기
- 회원가입
    - 아이디 입력 (중복 확인)
    - 비밀번호 입력 (정규식 조건 있음)
    - 비밀번호 확인
    - 이메일 입력 (중복 및 정규식 확인)
    - 회원가입 버튼

---

### 회원

- 글로벌
    - 우하단 플로팅 글 작성 버튼
- 게시판 홈
    - 로그아웃
    - 마이페이지
    - 제목 검색
    - 게시글 [ 아이디 제목 생성일 ]
    - 페이지네이션 버튼
- 마이페이지
    - 내 정보 [아이디, 이름, 이메일, 생성일, 마지막 수정일]
    - 내 정보 수정 [이메일(중복확인 필요), 이름]
    - 내가 작성한 글 확인
    - 북마크 게시글 목록
    - 회원탈퇴
- 게시글 상세
    - 게시글 제목
    - 생성 날짜
    - 내용 (+이미지)
    - 댓글창
        - 댓글 작성 칸 [작성 버튼]
        - 작성된 댓글들 [ 생성일 작성자id 댓글 내용 ]
        - [댓글 작성자 본인] 내 댓글 삭제
    - 북마크 버튼 (활성화/비활성화 ui)
    - [작성자 본인] 수정 삭제 → 수정 시 화면이 수정 포멧으로 변경
- 게시글 작성 페이지
    - 제목
    - 내용
    - 이미지 선택 최대 3개
