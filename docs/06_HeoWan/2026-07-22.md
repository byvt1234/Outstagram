# [2026-07-22 09:24:50] 기초 공사 완료

- 기존에 존재하는 `X-Server` 프로젝트에서 있던 로그인/로그아웃, 회원가입, /me 새로운 스키마에 맞추어 리팩토링 하엿습니다.
- 포스팅 또한 이미지 `multipart/form-data`를 포함하여 업로드, 수정, 삭제, 조회가 가능하게 5개의 api 핸들러를 다시 생성하였습니다.
- mongoClient를 mongoose로 바꾼 뒤 Schema를 다시 만들어두었습니다.
- 공통 Api Error 처리기를 `/Server/102_utils/api/[ApiError | error.middleware.js]`에서 만들어서 `app.use(errorHandler)` 을 통해 예외를 `try-catch` 및 `throw new ApiError(status, message, errorData?)`를 통해 만들게 해주었습니다.
- 많은 부분에서 최소 검증 로직을 짜두었흡니다. (mongoose의 ObjectId가 충족되지 않거나 빈 `.trim()` 문자열, 게시글 소유자가 아님 등)

완료되어있는 API는 다음과 같습니다.
```
- `POST /api/auth/signup → authController.signup` (회원가입 구현됨, 명세는 /users. 이건 auth로 설정할 예정)
- `POST /api/auth/login → authController.login` (로그인 구현됨)
- `GET /api/auth/me → authController.me` (JWT 필요)
- `GET /api/posts → postController.getPosts` (목록/페이지네이션 구현됨)
- `GET /api/posts/:id → postController.getPost` (상세 조회 구현됨)
- `POST /api/posts → postController.createPost` (multipart 이미지 업로드 포함)
- `PATCH /api/posts/:id → postController.updatePost` (기존 이미지 + 새 이미지 처리, 권한 실패 응답은 403 권장)
- `DELETE /api/posts/:id → postController.deletePost` (게시글 삭제 구현됨, 댓글/북마크/이미지 연쇄 삭제는 아직)
```
